# retail-sales-performance-dashboard
This project aims to derive actionable insights from sales data to understand product performance, customer behavior, and regional trends. The analysis covers best-selling product categories, monthly sales trends, most productive states, and underperforming categories.
Data Cleaning & Preparation
Handled missing values and inconsistencies.
Removed unnecessary characters and corrected data formatting.
Initial exploration in Excel, followed by detailed analysis using Python and SQL.
Key Insights
Identified the highest-priced item and the customer who purchased it.
Analyzed monthly sales performance, highlighting peak demand periods.
Determined top-performing and underperforming product categories.
Revealed regional purchasing patterns, showing which products are most popular in each state.
Recommendations
Inventory Management: Adjust stock levels to prevent shortages during peak periods.
Boost Underperforming Products: Investigate declining categories and implement marketing strategies (pricing, promotions, coupons).
Customer Retention: Introduce loyalty programs in states with the largest customer base to increase satisfaction and retention.
Tools Used
Excel for initial exploration
Python for data analysis and visualization
